A MIDI CC device developed with a Pro Micro.

---

### Tool used:
- VSCode with PlatformIO extension
- KiCad for the schematic

---

### Parts used:

- Pro Micro controller
- 4067 MUX
- 14 potentiometers
- one button for reseting the Pro Micro
